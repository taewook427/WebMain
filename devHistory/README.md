### Development History

Shows repositories and relationships between them.

| Field | Explanation | 설명 |
| :-- | :-- | :-- |
| domain | Classification of code | 코드의 대략적인 분류 |
| generation | Technology Level or Design Philosophy | 기술 수준이나 설계 사상의 분류 |
| project | A collection of repositories under the same purpose | 동일한 목적 하에 있는 레포들의 집합 |
| node | Single repository or Topic | 단일 레포지토리 혹은 주제 |
| edge | Indicates the previous affected node | 영향을 받은 이전 노드 표시 |

**Please make PR whenever you want to add or edit data!**
